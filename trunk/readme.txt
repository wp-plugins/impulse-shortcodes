Wordpress plugin to add shortcodes for Twitter Bootstrap 3.0 and FontAwesome 4.0

== Description ==

Impulse Shortcodes offers shortcodes for Twitter Bootstrap components and FontAwesome icons.

Please report issues [here](https://www.impulse-themes.com/forums/forum/impulse-shortcodes/).

== Installation ==

1. Unzip files.
2. Upload the folder into your plugins directory.
3. Activate the plugin.
4. Add new shortcodes to posts or pages.
